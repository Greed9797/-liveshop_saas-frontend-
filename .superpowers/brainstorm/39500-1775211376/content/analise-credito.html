✅ AUTO-APROVAÇÃO (Anti-Dopamina)

  • Status: ativo · Pin: 🟢 · Dashboard atualiza
  • Franqueado vê roleta girar sem esperar
❌ CAMINHO CRÍTICO
  • Score < 60 → backoffice analisa manualmente
  • "Assumir Risco" → modal com checkbox obrigatório
  • BD: is_risco_franqueado = true
  • "Arquivar" → status cancelado · pin 🩶 cinza
🔄 BATCH AO APROVAR

  • Dashboard polling 15s detecta → roleta gira
